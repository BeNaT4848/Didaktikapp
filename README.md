# Didaktikapp
Aplicación desarrollada junto al alumnado de Educación Infantil de la Universidad de Leioa. Ellos aportaron las necesidades educativas y contenidos, y nuestro equipo diseñó y programó la app. Está dirigida a niños de 4-5 años e incluye juegos y actividades en distintas ubicaciones para aprender de forma divertida e interactiva.
