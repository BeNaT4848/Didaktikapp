Aplicación Educativa Infantil

Este proyecto es una aplicación educativa desarrollada en Kotlin, dirigida a niños y niñas de 4 a 5 años. Su objetivo principal es ofrecer una experiencia de aprendizaje interactiva y divertida mediante juegos educativos adaptados a su nivel.

- Características principales
- Juegos educativos interactivos: actividades diseñadas para fomentar la memoria, la lógica, la atención y el aprendizaje básico.
- Uso de hilos (Threads): implementación de procesos en segundo plano para mejorar el rendimiento y permitir múltiples acciones simultáneas en los juegos.
- Base de datos local: gestión de datos de usuarios y progreso mediante almacenamiento local.
- Gestión de usuarios: sistema de registro, inicio de sesión y seguimiento del progreso de cada usuario.
- Personalización visual: sistema de paleta de colores personalizada para adaptar la interfaz a diferentes preferencias.
- IA integrada: asistente inteligente que permite a los usuarios realizar preguntas y recibir respuestas dentro de la aplicación.
  
Tecnologías utilizadas
Kotlin
Base de datos local (SQLite / Room)
Hilos (Threads / Coroutines)
Arquitectura orientada a componentes
Integración de API de IA
🎯 Objetivo del proyecto

El objetivo de esta aplicación es combinar la educación infantil con la tecnología, creando un entorno seguro, interactivo y atractivo donde los niños puedan aprender jugando, mientras se adapta la experiencia a cada usuario.
